How to play?

Color Game is a simple memory game that uses 8 colored buttons that each correspond to notes on the major scale.
When you press the start button the game will begin.
A pattern will be displayed on screen and once it is finished , it's up to you to remember what the pattern was and to click on the buttons accordingly, in the exact order that you were shown.
Each time you correctly repeat the pattern you will get a point , and the pattern will start over but with one extra button added. In this way the pattern gets harder as you play the game.
